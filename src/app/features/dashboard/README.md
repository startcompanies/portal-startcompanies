# Dashboard - Panel Administrativo

Esta carpeta contiene el panel administrativo para clientes (en desarrollo).

## Estructura

- `layout/` - Componentes de layout (sidebar, header del panel)
- `pages/` - Páginas del panel (dashboard-home, profile, orders, documents, settings)
- `guards/` - Guards de autenticación específicos del panel
- `services/` - Servicios específicos del panel

## Estado

Actualmente en preparación. La estructura está lista para comenzar el desarrollo.
